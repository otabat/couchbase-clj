couchbase-clj.config documentation

default-bucket
Default bucket name.
You can change the value by using swap! or reset!.
Default value is set to "default".
default-data-expiry
Default expiry for the data (in seconds).
You can change the value by using swap! or reset!.
Default value is set to -1.
default-inc-offset
Default offset for incrementing the data.
You can change the value by using swap! or reset!.
Default value is set to 1.
default-lock-expiry
Default expiry for locking the key (in seconds).
You can change the value by using swap! or reset!.
Default value is set to 15.
default-password
Default bucket password.
You can change the value by using swap! or reset!.
Default value is set to "".
default-uris
Default URI sequence collection of one or more servers from the cluster.
You can change the value by using swap! or reset!.
Default value is set to ["http://127.0.0.1:8091/pools"].
